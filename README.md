## U G F (U Got Film)

### 영화 추천 큐레이션 웹 서비스

---

국내외 현재 상영작 및 인기작 추천 외 유저의 기본 정보를 바탕으로 연령 및 성별에 맞추어 영화를 추천하는 종합 무비 큐레이션 웹 서비스

Tmdb api를 사용한 실시간 데이터 활용 및 서버리스 개발
